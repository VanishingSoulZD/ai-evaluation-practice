# Iteration 004 - Day 4

## 任务概述

**任务名称**：数据分布可视化  
**所属周**：Week 1 - 数据标注与数据分析基础  
**目标**：掌握标注数据可视化方法，了解数据分布特征，为后续分析和汇报提供直观图表。

## 背景/目的

- 理解数据在不同维度上的分布情况（分类标签、实体标签、BIO 标签等）
- 掌握 Python 可视化工具（Matplotlib、Seaborn）基本使用
- 提高数据分析和汇报能力，让标注结果直观展示

## 要解决的问题

- 数据标签分布不直观，难以快速评估标注情况
- 无法直观发现异常或不平衡的类别
- 需要将数据统计结果转化为可汇报图表

## 参考资料

- Matplotlib 官方教程（https://matplotlib.org/stable/tutorials/index.html）
- Seaborn 官方教程（https://seaborn.pydata.org/tutorial.html）
- Day 2 标注生成的 CSV/Excel 文件：`day2_annotation_practice.csv` / `day2_annotation_practice.xlsx`
- Day 3 一致性分析报告（参考分析指标和数据清洗方法）

## 任务要求

- 使用 Python 导入 Day 2 标注数据
- 可视化不同任务类型标签分布（如 POS/NEG/NEU）
- 可视化实体标签出现频率（PER/ORG/LOC/TIME）
- 可选绘制 BIO 标签序列或异常分布热力图
- 图表保存为 PNG 或输出到报告目录
- 输出总样本数及各类数量统计，便于汇报使用

## 实现步骤

1. 导入 Day 2 标注生成的 CSV 或 Excel 文件
2. 检查数据完整性（缺失值、重复样本）
3. 使用 Matplotlib / Seaborn 绘制图表：
   - 直方图：文本分类标签分布
   - 条形图：实体标签出现频率
   - 热力图（可选）：BIO 标签分布或异常统计
4. 设置图表标题、坐标轴标签、图例及颜色方案，使图表清晰易读
5. 保存图表文件到 `reports/` 或指定目录
6. 输出总样本数及每类标签数量统计表格（可 CSV/Excel 保存）

## 验收标准

- 图表清晰展示数据分布，包括分类、实体和 BIO 标签
- 图表保存完整，可直接用于汇报或分析
- 数据总量和各类统计与 Day 2 标注内容一致
- Python 脚本可重复运行，对不同批次标注数据适用
- 可选图表如热力图需反映异常或不均衡标签分布
