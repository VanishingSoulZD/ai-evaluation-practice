# Iteration 005 - Day 5

## 任务概述

**任务名称**：小型综合练习 — 标注、分析与可视化  
**所属周**：Week 1 - 数据标注与数据分析基础  
**目标**：将前四天的标注与分析技能结合，完成 50 条样本标注，并生成数据分布统计和可视化图表，实现完整的“标注 → 分析 → 可视化”流程。

## 背景/目的

- 将数据标注、数据一致性检查和可视化能力整合应用
- 熟悉从原始文本到可视化报告的完整流程
- 为后续更大规模数据处理和模型训练打基础

## 要解决的问题

- 学会在一个小型数据集上完成完整的数据标注流程
- 掌握统计标注一致性、缺失值、重复值的方法
- 能够生成清晰的分布图表，用于展示标注数据特点

## 参考资料

- 开源文本数据集：SQuAD、CoQA
- 自行模拟的小型数据集（50 条样本）
- Day 1–4 标注规则文档及可视化脚本：`day1_annotation_practice.md`、`day2_annotation_practice.md`、`day4_visualization.py`
- Pandas、Matplotlib、Seaborn 官方教程

## 任务要求

- 完成 50 条样本标注
- 使用 Python 对标注数据进行统计分析，包括：
  - 分类标签分布（POS/NEG/NEU）
  - 实体标签统计（PER/ORG/LOC/TIME）
  - BIO 序列统计
  - 缺失值和重复值检查
- 绘制可视化图表：柱状图、热力图等
- 输出完整统计表格（CSV/Excel 可选）和图表文件
- 将标注、统计、可视化流程整合，形成可复现报告

## 实现步骤

1. **数据标注**
   - 准备 50 条文本样本
   - 使用 Day 1–2 的标注规则进行标注
   - 记录标注日志，包括样本 ID、原文、标注结果和发现问题

2. **数据分析**
   - 使用 Pandas 导入标注数据
   - 统计各类标签数量及分布
   - 检查缺失值和重复值
   - 计算标注一致率（可选：与参考答案或复标结果对比）

3. **数据可视化**
   - 使用 Matplotlib / Seaborn 绘制：
     - 文本分类标签分布柱状图
     - 实体标签出现频率柱状图
     - BIO 标签分布热力图
   - 保存图表到指定目录
   - 生成统计表格 CSV/Excel 文件

4. **报告整理**
   - 将标注日志、统计表格、可视化图表整合成一个小型报告
   - 确保流程可复现，便于其他数据集重复使用

## 验收标准

1. 完成 50 条样本标注，记录完整
2. 输出统计表格（CSV/Excel）包含：
   - 样本总数
   - 缺失值数量
   - 重复值数量
   - 分类标签分布
   - 实体标签分布
   - BIO 标签分布
3. 输出可视化图表（柱状图、热力图），清晰展示标签分布
4. 能顺利执行完整流程：“标注 → 分析 → 可视化”，结果完整且正确
