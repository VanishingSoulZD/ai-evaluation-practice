## 任务详细说明

### 1. 汇总脚本

- 将 Day 15~Day 20 的主要模块整理到统一目录：
  - `main.py`：API 接口整合
  - `tasks.py`：异步任务整合
  - `week2_metrics.py`：指标计算模块
- 删除冗余文件，保留最终可运行版本
- 确保 UPLOAD_DIR、REPORT_DIR 等目录创建逻辑统一

### 2. 编写使用说明文档

- 内容包括：
  - 安装依赖（pyproject.toml / pip 安装命令）
  - 启动 FastAPI 服务命令
  - 启动 Celery Worker 命令
  - 接口调用说明：
    - `/upload-file`
    - `/batch-metrics`
    - `/batch-metrics-async`
    - `/batch-report-async`
    - `/process-csv-async`
    - 任务状态查询接口
  - 示例请求和返回结果
  - 异常处理说明

### 3. 验证接口和流程

- 测试上传 CSV 文件
- 测试同步指标计算
- 测试异步指标计算及报告生成
- 查询任务状态，确认返回结果正确
- 检查生成 CSV 文件和图表报告

### 4. 代码规范与注释

- 确保所有函数、接口、任务有中文注释
- 保证变量命名统一
- 目录结构清晰，便于后续复用

### 5. 输出结果

- 生成最终文档（README.md / 使用手册）
- 确认代码可直接运行，接口调用无报错
- 将整理后的项目提交至版本控制或归档
